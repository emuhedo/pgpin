## CLI

Get overview help:

```console
$ datapins
Usage: datapins <command> [<args>]

Commands:
  list     List datapins
  create   Create a new datapin
  destroy  Destroy a datapin
  help     Get help for a command
....

List dataclips:

````
$ datapins list

Creating datapins... done
https://datapins.heroku.com/

$ datapins:create
